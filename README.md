# micoli

mini mini (about 1000 lines of code) coding agents 🤖

## Getting Started

```bash
export OPENROUTER_API_KEY=your-key-here
npx micoli
```

Get your API key at [openrouter.ai/keys](https://openrouter.ai/keys).

### Requirements

- `Node.js`
- `Docker`
- `ripgrep`

## Features

```md
- Coding agent with minimal tools
  - Read `AGENTS.md`
  - File tools:
    - list
    - find
    - read
    - edit
  - Exec tools:
    - exec: It runs on Docker container
  - Web tools:
    - fetch
```

## Development

```bash
npm install
npm start
```
